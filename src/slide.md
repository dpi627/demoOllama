# .git**ignore**
![bg right:40%](https://picsum.photos/720?image=102)
- 檔案名稱就叫 `.gitignore`
- 與 `.git` 目錄放在一起
- 可以使用任何**文字編輯器**產生或編輯
- 排除不需要版本控制的檔案或目錄，例如**編譯生成檔案**、**下載套件**、**個人設定檔**、**臨時性文件**、**過大的檔案**或**日誌文件** ... 等等
- [gitignore.io](https://www.toptal.com/developers/gitignore/) 線上生成網站
- 多數服務已於建立 Repo 時提供此功能
